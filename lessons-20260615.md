## FEEDBACK — Armadilhas e Erros

### 1. `ast.alias.names` não existe (é `.name`)
Ao tentar verificar imports não utilizados via AST introspection, errei o atributo — `alias` não tem `.names`, é `.name`. Verificar imports manualmente com `py_compile + grep` é mais confiável.

### 2. GitHub API retorna 404/403 se features desabilitadas
Dependabot e CodeQL retornam 403 se não habilitados no repositório. Verificar antes de depender da API.

### 3. `pip install` no MacOS sem venv falha (PEP 668)
`python3 -m pip install` sem venv no MacOS bloqueia com `--break-system-packages`.
**Solução**: `python3 -m venv .venv && source .venv/bin/activate && pip install ...`

### 4. Planos que mexem nas mesmas linhas exigem ordem de execução
Plans 004 (bare except) + 007 (style) mexem nas mesmas linhas. Plans/README.md com dependency graph resolveu.

### 5. Drift check salvou de executar planos quebrados
`git diff --stat <base>..HEAD -- activate.py` no início de cada plano.
Manter como padrão em todos os planos futuros.

### 6. STOP conditions impedem execução cega
Essencial para planos executados por outros agentes/executores. Sempre incluir.
